<h2 align="center">Description</h2>
<p align="center">
    In my JavaFX project, I have seamlessly integrated API calls to retrieve data from my colleagues' backend. The user interface, crafted using SceneBuilder and enriched with CSS styling, boasts a polished appearance.
    One notable feature of my application involves interaction with a CAEN RFID device, facilitating the streamlined scanning of merchandise tags. By connecting the RFID device to my application, the need for manual input of RFID codes for each item has been eliminated. Instead, scanning the tags using the connected device populates the information tab for the asset effortlessly.
</p>
